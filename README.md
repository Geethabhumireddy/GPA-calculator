# SGPA/CGPA Calculator

This project is a student-made web application designed to calculate Semester SGPA (Semester Grade Point Average) and CGPA (Cumulative Grade Point Average) specifically for students of BITM.

## Features

- **SGPA Calculation:** Allows students to input their credits and marks for each subject to calculate their SGPA.
- **CGPA Calculation:** Allows students to input their semester GPAs to calculate their overall CGPA.
- **User-Friendly Interface:** Simple and intuitive design for easy navigation and input.
- **Modal Display:** SGPA results and CGPA results are displayed in a modal for better visibility.

## Technologies Used

- **HTML:** Structure of the web application.
- **CSS:** Styling for the web application.
- **JavaScript:** Logic for calculating SGPA and CGPA to displaying results.
